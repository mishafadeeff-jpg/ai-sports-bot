# ⚽ AI Sports Analyst Bot (Telegram)

Полнофункциональный Telegram-бот ИИ-Аналитики ставок на спорт с системой ручной оплаты по СБП и встроенной админ-панелью для управления подписками.

## 🚀 Быстрый запуск

1. Запустите файл `start_bot.bat` двойным кликом на компьютере.
2. Бот автоматически установит зависимости и запустит процесс long-polling.

---

## ⚙️ Настройка реквизитов СБП и бота (`config.py`)

Откройте файл `config.py` в любом текстовом редакторе и укажите ваши данные:

```python
# 1. Токен бота от @BotFather
BOT_TOKEN = "ВАШ_ТОКЕН_БОТА"

# 2. Ваш личный Telegram ID (чтобы получаться чеки и управлять ботом)
ADMIN_IDS = [720963162, 8791637436]

# 3. Ваши реквизиты СБП для приема переводов
SBP_PHONE = "+7 (999) 000-00-00"
SBP_BANK = "Т-Банк / Сбербанк"
SBP_RECIPIENT = "Михаил Ф."
```

---

## 💎 Как это работает для вас (Админа)

1. **Клиент нажимает «Купить VIP» в боте.**
2. Бот выдает ему ваши реквизиты СБП и просит прислать скриншот чека.
3. Клиент переводит деньги на вашу карту и скидывает фото чека в бот.
4. **Бот присылает фото чека вам в личку** с двумя кнопками: `[✅ Подтвердить]` и `[❌ Отклонить]`.
5. При нажатии на **`[✅ Подтвердить]`**:
   - Бот сам рассчитывает срок подписки и активирует VIP-доступ клиенту.
   - Клиенту приходит радостное уведомление в чат.
6. Вы можете проверить общую статистику доходов и пользователей командой `/admin`.
